# DriveSafe Rover Firmware

This folder contains the firmware for the ESP32-based DriveSafe Rover.

## Features
- Manual and Autonomous Driving
- Ultrasonic Obstacle Detection
- GPS Location Tracking
- ESP32-CAM Wi-Fi Streaming
- Headlight, Tail Light & Hazard Light Control
- Servo-Based Camera and Sensor Scanning
- Web Dashboard Communication

## Platform
- ESP32 Dev Module
- Arduino IDE
